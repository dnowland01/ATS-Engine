# ATS Engine (Pure PHP)

A fully automated job application tracking engine built in pure PHP 8.2+.  
It processes job descriptions, submissions, failures, and interview invites,  
and automatically updates two CSV tables:

- `main_table.csv` (Fit ≥ 93%)
- `under93_table.csv` (Fit < 93%)

Includes:

- CLI interface  
- Web dashboard  
- Daily summary logging  
- Duplicate detection  
- Status + notes automation  
- Source + agency detection  
- Docker support
